BOIII v1.1.7 - NO-UPDATE PACKAGE
================================

This minimal package contains only:

  * boiii-v1.1.7.exe - the verified BOIII v1.1.7 client
  * launch-boiii-1.1.7.bat - launches it with -noupdate
  * this instruction file

It does not contain any zm8 GSC scripts, Lua scripts, or gameplay changes.

INSTALL
-------

1. Extract all three files into your Black Ops III folder.
2. Always start BOIII with launch-boiii-1.1.7.bat.
3. Do not launch through an updater if you want to remain on v1.1.7.

The -noupdate argument prevents this preserved v1.1.7 executable from
contacting the update service and replacing itself with a newer build.
